ErgoActive Download

ErgoActive runs on Microsoft Windows 95/98/ME/NT/2000/XP.

To download ErgoActive version 1.0:

  1. Click here to download the ErgoActive set-up file (1.1 megabytes)
  2. When downloaded execute it
  3. You will then be guided through the set-up process

NOTE: The installation requires you to be logged on as a user with administrator level access rights on Windows NT, 2000 or XP.
